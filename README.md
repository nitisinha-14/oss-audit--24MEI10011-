## 📌 Project Description

This project is based on analyzing an open-source software system and understanding its technical and philosophical aspects.

For this audit, I selected Git because it is widely used in software development and plays a key role in collaboration and version control.

The project includes:

* A detailed report covering origin, license, ethics, and ecosystem
* Practical implementation using Linux commands
* Five shell scripts demonstrating basic automation and system understanding

---

## ⚙️ Requirements

Make sure the following are installed on your Linux system:

* Git
* Bash Shell
* Linux Distribution (Ubuntu recommended)

---

## 🛠️ Setup Instructions

### 1. Install Git

```bash
sudo apt update
sudo apt install git
```

### 2. Verify Installation

```bash
git --version
```

```bash

```

### Step 2: Run Scripts

```bash
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog
./script5.sh
```

## 📜 Script Descriptions

### 🔹 Script 1 — System Identity Report

Displays system details like kernel version, user, uptime, and OS information.

---

### 🔹 Script 2 — FOSS Package Inspector

Checks if Git is installed and shows its version and purpose.

---

### 🔹 Script 3 — Disk and Permission Auditor

Analyzes important system directories and displays their size and permissions.

---

### 🔹 Script 4 — Log File Analyzer

Reads a log file and counts how many times a keyword appears.

---

### 🔹 Script 5 — Manifest Generator

Takes user input and generates a personalized open-source philosophy statement.
